Всего есть 5 различных кастов: static_cast, const_cast, reinterpret_cast, dynamic_cast, C-style-cast.
# static_cast
Во всех преобразованиях, где нужно например из int сделать double используют **static_cast**. При этом происходит перекодировка.
Используется для явного приведения типов, чтобы избежать каких-либо конверсий.

```cpp
// сначала в какой тип кастим, а потом что
double a = static_cast<double>(123521);
// возвращает переменную с таким же значением, но другого типа
```

Из очень полезных опций: **static_cast** может приводить указатели к разным типам (полезнее всего к void*).

# reinterpret_cast
Это способ взять биты одной переменной и трактовать их как биты переменной другого типа, без перекодирования. Т.е. буквально был int64_t, а теперь все записанные биты трактуются как double.

```cpp
int64_t y = 1235213423433243234LL;
std::cout << reinterpret_cast<double&>(y); // строго говоря UB
// вывод 4.29718e-226
```

**reinterpret_cast** не позволяет создавать новые объекты, но позволяет только читать биты в памяти по-другому, поэтому возвращаемый тип только ссылка нужного типа.
Это бывает полезно и безопасно, когда, например в структуре, полностью совпадают все типы полей и их порядок, но неопределенно преобразование из одной структуры во вторую.
Также **reinterpret_cast** позволяет кастовать любые указатели к любым, даже если они несовместимые.

**reinterpret_cast** не может приводить статические переменные к указателям и наоборот.
Также ему запрещено просматривать константы.

# const_cast
Специальный каст чтобы отменить константность. Это нужно для ситуаций, когда дали константную ссылку, но мы знаем, что переданное значение изначально не было константным. Для изначальных констант каст к обычному типу это UB.
Также как и reinterpet_cast, const_cast не создает новую копию, только ссылку и в целом это как reinterpert_cast, только для отмены константности.

# C-style-cast
Это все выше перечисленное. Любой ценой приведет один тип к другому, если не одним кастом, так комбинацией и только если во всех случаях будет ошибка компиляции выкинет ошибку.
