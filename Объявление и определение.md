# Объявление и определение

## Объявление

Объявление позволяет объявлять сущности, но не задавать значение.
Объявлять сущности можно сколько угодно раз.
Глобально объявлять можно переменные, функции, структуры, классы, пространства имен, алиасы (юзинги).

```C++
#include <iostream>

int x; // объявлена глобальная переменная х

void f(int x); // объявлена функция f

class C; // объявлен класс C

int main() {}
```

Объявление может быть разным
#### Скоуп
Скоуп по сути это область видимости внутри фигурных скобок.
Скоуп можно создавать в любой части кода, достаточно указать фигурные скобки.

```C++
#include <iostream>

int main() {
	int x = 10;
	{ // начало локального скоупа
		int x = 20;
		std::cout << x << '\n'; // 20
	}	
}
```

Заметим, что объявление сущностей в скоупе является приоритетным.

### Глобальное
Это значит что сущность объявлена в глобальной области видимости. Глобальный это значит что он лежит нигде, т.е. на самой верхушке скоупа.

Глобальный скоуп лежит в верху программы:
```C++
#include <iostream>

int x = 5; // глобальная переменная

// глобальный скоуп

int main() {
	// локальный скоуп main
}
```

Глобальная область видимости позволяет обращаться к глобально объявленным сущностям из любого места в коде.
### Локальное
Это все объявление внутри определенного блока кода.
Любой скоуп называется локальным, если он ограничен фигурными скобками.

```C++
#include <iostream>

void f() {
	// локальный скоуп f
}

int main() {
	// локальный скоуп main
	{
		// это тоже локальный скоуп
	}
}
```

## Определение

Для определения существует правило: Для каждого объявления существует только одно определение. Любое определение является объявлением
Это не совсем работает с классами. Классы могут быть несколько раз определены, если это дословное определение.
Правило одного определения не работает для namespace, потому что когда мы пишем ключевое слово и имя пространства подразумевается, что мы в него дописываем.

Определение начинается с создания скоупа для объявленной сущности (т.е. открытием фигурных скобок)
